# Zadatak 1: Rad sa funkcijama u Pythonu

Dobrodošli u vaš prvi zadatak! Vaš cilj je da dovršite kod u fajlu `zadatak.py`.

## Uputstvo za rad:
1. Otvorite fajl `zadatakKvadrat.py`.
2. Pronađite funkciju `izracunaj_kvadrat` i zamenite reč `pass` kodom koji stvarno računa kvadrat broja.
3. U glavnom delu programa zamenite `"???"` pozivom vaše funkcije.
4. Kada završite, uradite `commit` i `push` vaših izmena nazad na GitHub.

**Rok za predaju:** 7 dana od dodele zadatka.
